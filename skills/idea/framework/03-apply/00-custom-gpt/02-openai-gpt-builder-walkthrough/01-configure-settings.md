# SKILL — Configure Advanced Settings (Builder Step 3)

## Purpose
Once the basic GPT structure exists, the Configure tab unlocks the customization that makes it feel like a branded product: profile picture, fine-tuned instructions, and conversation starters.

---

## Step 3: Configure Advanced Settings
Once the basic structure is in place, click **Configure** to access additional customization options.

### Update Profile Picture
- Upload an image from your device.
- Or use **DALL·E** to generate a custom AI avatar based on textual input.

### Fine-Tune Instructions
- Modify initial response guidelines to shape your GPT's tone and behavior.
- Provide examples of how it should (or shouldn't) answer questions.

### Set Up Conversation Starters
- Add default prompts users can click to initiate interactions.
- Remove any suggestions that don't align with the GPT's purpose.

## Coaching Application
The "how it should (or shouldn't) answer" examples here are the UI home for the system-instructions template's Example Interaction and "what NOT to say" pairs — port them in. Curate conversation starters to the brand's top use cases; leftover generic starters dilute the branded feel.
