# SKILL — Cohesive Voice Overview (The Four Elements)

## Purpose
Names the four elements that combine into one consistent brand personality: tone, language, design, and imagery. Use this as the bridge between the IDEA pillars (the strategy) and the executional elements (the craft).

---

## Voice Is a Holistic Experience
> "Brand voice extends beyond words—it is a holistic experience that shapes how customers perceive and connect with a brand. When tone, language, design, and imagery align seamlessly, they create a consistent brand personality that strengthens customer relationships and influences purchasing decisions."

## The Four Elements
A cohesive brand voice is built from four aligned elements:

| Element | What it carries |
|---|---|
| **Tone** | the emotional signature — how the brand speaks |
| **Language** | the distinctive expression — taglines and copy |
| **Design** | the visual language — color, typography, layout |
| **Imagery** | the visual reinforcement of the emotion |

When these align, the result is a **consistent brand personality** that strengthens relationships and influences purchasing decisions.

## Coaching Application
Treat the four elements as a checklist for consistency. After the seller has set strategy via the IDEA pillars, confirm each element is pulling in the same direction: does the tone, the language, the design, and the imagery all express the same personality? A mismatch anywhere (e.g., bold copy on a timid visual) breaks the cohesion that drives purchasing decisions.
