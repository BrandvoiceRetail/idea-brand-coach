# SKILL — Brand Statements Canvas Table

## Purpose
The third table of the worksheet, written AFTER the Avatar 2.0 and IDEA Framework tables are complete. A four-column grid — "Brand Statements | Description | Example | Your Statement" — with five rows: brand purpose, brand vision, brand mission, brand values, and target audience. Each Description is the IDEA-rooted definition; each Example is the worked sustainable/eco-brand sample; each "Your Statement" is the fill-in. Reproduced verbatim from the worksheet — copy it exactly.

---

## Coaching Application
Only start this table once the Avatar 2.0 and IDEA Framework tables are filled — the Descriptions tell the user (and the AI) exactly what each statement must do, and the Examples show the standard. Note the worked Examples all belong to one coherent eco/sustainable brand: keep the user's five statements equally coherent with each other, not five disconnected sentences. For the Target Audience row, pull directly from the Avatar 2.0 outputs rather than writing a fresh demographic guess. The per-element breakdowns (purpose, vision, mission, values, target audience) add the four Key IDEA Principles and an AI prompt for each row.
