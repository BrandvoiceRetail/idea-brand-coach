# SKILL — IDEA Framework Brand Canvas Table

## Purpose
The second table of the worksheet: "IDEA Framework Brand Canvas: Capturing Customer-Centric Insights." A two-column "Framework Pillar | Insights" grid with one fill-in prompt for each of the four IDEA pillars. Reproduced verbatim from the worksheet — copy it exactly when building the user's canvas.

---

## The Table (verbatim)

**IDEA Framework Brand Canvas: Capturing Customer-Centric Insights**

| Framework Pillar | Insights |
|---|---|
| Insight-Driven | Record your insights here. |
| Distinctive | Describe how your brand is Distinctive/Different/Salient |
| Empathetic | How do we align with customers' personal beliefs and life goals? Describe exactly how we "get them". |
| Authentic | What is the source of our authority/credibility in the space? Professional experience? Expertise? Studies? Research? Social Proof? |

## Coaching Application
Fill all four pillar rows before moving to the Brand Statements table. Each prompt is a probe: "Insight-Driven" pulls the customer understanding from Chapter 2's Insight pillar; "Distinctive" forces a salience/difference answer; "Empathetic" demands a concrete "here's exactly how we get them"; "Authentic" requires naming a real credibility source (experience, expertise, studies, research, or social proof) — not a vague claim. If the Authentic box has no concrete source, the brand has an authenticity gap to fix before writing statements.
