# SKILL — IDEA Customer Avatar 2.0 Canvas Table

## Purpose
The first table of the worksheet: a 7-row capture grid that records the Avatar 2.0 outputs (from Chapter 3) into the canvas. Each question is paired with a "Record here" box the user fills in. This table is reproduced verbatim from the worksheet — copy it exactly when building the user's canvas.

---

## The Table (verbatim)

| IDEA Customer Avatar 2.0 | |
|---|---|
| Buyer intent / purpose of the product / function? | Record here |
| Motivation for the purchase / what do they want to achieve? | Record here |
| What are the underlying emotional triggers / drivers for the purchase? | Record here |
| What is their emotional state pre-purchase (how do they feel)? | Record here |
| What is their emotional state post-purchase (how do they want to feel)? | Record here |
| What Shopper Type are they? | Record here |
| Relevant demographics | Record here |

## Coaching Application
Walk the user through all seven rows in order before moving to the IDEA Framework table. Each row maps directly to a Chapter 3 Avatar 2.0 concept — buyer intent, motivation, emotional triggers, the pre/post emotional states, Shopper Type, and demographics-as-context. If a "Record here" box is still empty or vague, send the user back to the relevant Avatar 2.0 work rather than letting them guess — these answers feed the brand statements downstream.
