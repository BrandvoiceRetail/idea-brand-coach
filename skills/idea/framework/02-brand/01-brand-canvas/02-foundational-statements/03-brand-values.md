# SKILL — Brand Values

## Purpose
The fourth brand statement: the guiding principles that shape how a brand interacts with customers, employees, and stakeholders. Captures the definition, the four Key IDEA Principles, the AI prompt, and the worked sample output — all verbatim.

---

## Definition
Brand values are the **guiding principles that shape how a brand interacts with customers, employees, and stakeholders.** They create a foundation for consistency, trust, and emotional connection.

## Key IDEA Principles
- **Insight-Driven:** Reflects what customers value in a brand beyond products.
- **Distinctive:** Sets the brand apart through a strong moral or ethical standpoint.
- **Empathetic:** Aligns with customer beliefs, fostering loyalty and advocacy.
- **Authentic:** Ensures the brand's actions consistently reflect its promises.

## AI Prompt (Example)
> "Identify the core values that define the brand's character and ethical principles. Ensure these values reflect customer expectations, create an emotional connection, and set the brand apart in its industry. Use the IDEA framework to make them distinctive, empathetic, and authentic. Provide examples of how these values are consistently demonstrated in branding, messaging, and operations."

## Sample Output
> "Integrity, innovation, sustainability, and community. We commit to ethical practices, groundbreaking ideas, and fostering connections that benefit the planet and its people."

## Coaching Application
Values are only credible when they show up in action — the Authentic principle is "ensures the brand's actions consistently reflect its promises." Push the user past a generic list of adjectives: the sample pairs each value-set with a commitment ("We commit to ethical practices, groundbreaking ideas, and fostering connections"). When the AI returns values, ask the user to name one concrete way the brand already demonstrates each — if they can't, the value is aspirational, not real.
