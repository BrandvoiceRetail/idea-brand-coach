# Integrations

Operator + developer guides for third-party integrations in IDEA Brand Coach. Each
integration follows the same shape: server-side OAuth via Supabase Edge Functions,
tokens stored service-role-only (never client-readable), and a connection card on the
in-app Integrations page. See **[CANVA.md](./CANVA.md)** for the Canva Connect
integration (link a Canva account, list designs, import into the Brand Coach) —
including Canva developer-app setup, the four function secrets, deploy commands, and
troubleshooting. A sibling **Figma** integration is in progress in another worktree and
will get its own guide here when it lands.
